## Our Mission

To make powerful technology accessible to small businesses. We close the gap between enterprise-grade software and the businesses that need it most — delivering affordable, AI-driven tools that solve real problems and create real value.

## Live Products

### Smart Scheduler — AI-Powered Appointment Booking
Multi-tenant SaaS platform with AI conversational scheduling, Twilio voice support, Stripe billing, and AWS deployment.
- **Stack:** Python, FastAPI, OpenAI GPT-4o, Twilio, Stripe, SQLite, AWS ECS Fargate
- **Live:** [ai-scheduling-agent.com](https://ai-scheduling-agent.com)

### Wager Up — Skill-Based Wagering Platform
Real-time wagering platform for golf, esports, basketball, football, darts, and bowling.
- **Stack:** React 19, Tailwind CSS, Framer Motion, FastAPI, MongoDB, Socket.IO, Stripe
- **Live:** [wagerupapp.com](https://wagerupapp.com)

### Token Saver — AI Prompt Optimization
Chrome extension for real-time token counting, efficiency scoring, and one-click prompt optimization.
- **Stack:** JavaScript, Chrome Extension (MV3), js-tiktoken, Webpack, OpenAI API
- **Demo:** [Live Demo](https://todddunn84-ux.github.io/Token-Saver/)
